# Лабиринт сокровищ
# Текстовая игра, где игрок исследует лабиринт, собирает предметы, решает загадки и ищет сокровища

# Способ 1
# Установка зааисимостей 
poetry install

# Запуск игры 
poetry run python main.py

# Способ 2 
# Установка
make install

# Запуск
make project

| Команда                 | Описание                                                |
| ----------------------- | ------------------------------------------------------- |
| `go <direction>`        | Перейти в указанном направлении (north/south/east/west) |
| `north/south/east/west` | Быстрое перемещение без слова `go`                      |
| `look`                  | Осмотреть текущую комнату                               |
| `take <item>`           | Поднять предмет из комнаты                              |
| `use <item>`            | Использовать предмет из инвентаря                       |
| `inventory`             | Показать содержимое инвентаря                           |
| `solve`                 | Попытаться решить загадку                               |
| `help`                  | Показать список команд                                  |
| `quit`                  | Выйти из игры                                           |


Описание комнат
1. entrance – Вход в лабиринт. Стены покрыты мхом, на полу старый факел. Выходы: north, east. Предметы: torch
2. hall – Большой зал с пьедесталом. Загадка: "Назовите число после девяти" (ответ: 10 или "десять"). Выходы: south, west, north
3. trap_room – Комната с плиточной ловушкой. Загадка: повторить "шаг" три раза подряд. Предметы: rusty_key. Выход: west
4. library – Пыльная библиотека. Загадка: "Что растет, когда его съедают?" (ответ: резонанс). Предметы: ancient_book. Выходы: east, north
5. armory – Старая оружейная комната. Предметы: sword, bronze_box. Выходы: south, north
6. secret_room – Тайная комната за старым шкафом. Предметы: treasure_key. Выход: south
7. treasure_room – Комната с сундуком. Дверь требует ключ или код. Предметы: treasure_chest. Выход: south. Загадка: "2*5=?" (ответ: 10)
8. underground_lake – Подземное озеро с серебряным ключом. Загадка: "Сколько букв в слове 'вода'?" (ответ: 4). Предметы: silver_key. Выход: west

Особенности игры
1. Случайные события: на каждом шаге есть шанс найти монетку, услышать шорох или попасть в ловушку
2. Ловушки: если игрок не вооружен факелом в trap_room, может потерять предмет или получить урон
3. Секретные комнаты: доступны только после решения загадок или использования специальных предметов
4. Псевдослучайность: все события детерминированы шагами игрока 
5. Инвентарь: игрок может поднимать, использовать и просматривать предметы, собранные по пути

Пример игрового процесса
1. Игрок начинает в entrance. Видит факел и может его взять: take torch
2. Идет в hall: go north. Решает загадку пьедестала: solve
3. Переходит в library или trap_room, исследует комнаты и собирает предметы
4. Находит rusty_key и treasure_key для доступа в treasure_room
5. Использует ключ или вводит код для открытия сундука: use treasure_chest или solve
6. Игра заканчивается победой после открытия сундука или поражением при попадании в ловушку без защиты
